# 🚀 The Date Lab - Portfolio Project

Chào bạn, mình là **Anh Vũ**! 👋 
Chào mừng bạn đến với **The Date Lab** - một ứng dụng web Full-stack do chính mình phát triển dành cho dịch vụ workshop dating & art therapy tại Hà Nội.

Dự án này là minh chứng cho khả năng xây dựng một ứng dụng web hiện đại từ Frontend, Backend đến tích hợp AI và thanh toán thực tế của mình.

## ✨ Điểm Nổi Bật Của Dự Án

- **Trải Nghiệm Người Dùng (UX/UI):** Giao diện hiện đại, mượt mà với các hiệu ứng chuyển động (Framer Motion), thiết kế responsive, hỗ trợ đa ngôn ngữ (VI/EN) và chuyển đổi giao diện linh hoạt.
- **Tính Năng Cốt Lõi:**
  - 🔍 Khám phá và tìm kiếm sự kiện theo "vibe" (phong cách).
  - 🎟️ Đặt vé tham gia workshop và quản lý đơn hàng.
  - 📸 Lưu trữ kỷ niệm (Upload hình ảnh).
  - 🔮 **Trải nghiệm Tarot:** Gợi ý sự kiện cá nhân hóa thông qua việc bốc bài Tarot.
  - 🤖 **Trợ lý ảo AI (Chatbot):** Tích hợp Google Gemini AI để hỗ trợ khách hàng, giải đáp thắc mắc và tư vấn sự kiện theo thời gian thực.
  - 💳 **Thanh toán:** Tích hợp VietQR API để tự động tạo mã QR chuyển khoản chính xác.
- **Hệ Thống Quản Trị (Admin Dashboard):** Quản lý sự kiện, vé, doanh thu, phân tích dữ liệu trực quan bằng biểu đồ (Recharts) và quản lý tài khoản ngân hàng nhận thanh toán.

## 🛠 Tech Stack Áp Dụng

Dự án được xây dựng với hệ sinh thái công nghệ hiện đại và ưu tiên hiệu năng:

- **Frontend:**
  - React 19 (Vite)
  - Tailwind CSS v4 & Framer Motion (UI/Animations)
  - Lucide React (Icons)
- **Backend & API:**
  - Node.js & Express
  - RESTful APIs với Authentication (JWT & bcryptjs)
  - Tích hợp AI: `@google/genai`
- **Database & Storage:**
  - Neon Serverless PostgreSQL
  - Drizzle ORM (Type-safe database manipulation)
  - Cloudinary (Quản lý và lưu trữ media)
- **Ngôn ngữ:**
  - TypeScript (End-to-end type safety)
